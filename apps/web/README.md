## Tech Stack

- **Framework**: React 18 + TypeScript
- **Build**: Vite
- **Routing**: React Router v6
- **State Management**: Zustand (with persist middleware)
- **Data Fetching**: React Query (@tanstack/react-query)
- **HTTP Client**: Axios
- **Styling**: Tailwind CSS
- **Icons**: Lucide React
